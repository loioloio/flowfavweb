Powerful Features

Over 30 tools designed to give you total control over your Kodi favorites.

List & Profile Management

  • Multi-Profile: Create unlimited profiles (independent JSON files) to separate environments like "Kids", "Sports", "Movies".
  • Import & Export: Easily share profiles or move them between devices using XML files.
  • System Templates: Predefined and editable lists with shortcuts to sections and useful Kodi commands.
  • Dynamic Widget: Specialized /widget path to integrate favorites as clean folders into widgets of advanced skins (Aura, Arctic, Titan, etc.).

Visual Editing & Customization

  • Pop-up Editor: User-friendly and fast interface to edit your favorites' properties.
  • Total Customization: Change text color, apply formatting (bold/italics) and adjust font and icon sizes.
  • Icon Management: Automatic icon assignment or manual selection of custom images.
  • Separator Bars: Insert divider lines or texts to visually organize your lists.
  • Accessibility: Interface scaling, high-contrast modes and adapted palettes.

Organization & Automation

  • Quick Sorting: Sort your favorites quickly and easily.
  • Bulk Operations: Multi-selection to move, delete or color items in groups.
  • Auto Grouping: Intelligent algorithm that groups favorites by their source addon with a single click.
  • Search & Filter: Sorting tools (A-Z, Z-A, Reverse) and real-time text filtering to quickly find content.

Creation & Manipulation

  • Manual Creation: Create favorites from scratch by typing the path, edit existing commands or duplicate entries.
  • Addon Selector: Integrated search to add favorites from any installed addon without leaving the editor.
  • Global Context Menu: Add content to Flow FavManager from anywhere in Kodi using the context menu.
  • Compatibility Maximizer: Automatically corrects problematic URLs to ensure all favorites execute correctly.

Security & Maintenance

  • PIN Protection: Lock access to the editor or profiles with a numeric code and intelligent session locking.
  • Rescue System: Recover your password via security question. As a last resort, create a reset_pass.txt file.
  • Audit Log: Internal log of critical actions for total control.
  • Backup System: Comprehensive tools to create backups and restore your favourites.xml file.

How to Install

Two simple methods to install Flow FavManager on your Kodi.

From Kodi File Manager

The simplest method. No external file downloads required and enables automatic updates.

Open Kodi and go to Settings (gear icon).

Enter File manager and select Add source.

Click on <None> and type this address exactly:
https://loioloio.github.io/Flow-FavManager/

Give the source a name (e.g. flowfav) and click OK.

Return to the main menu, enter Add-ons and click the open box icon (top left).

Select Install from zip file.

Choose the flowfav source and select repository.flowfavmanager-1.0.0.zip.

Wait for the "Add-on installed" notification (this installs the repository).

In the same menu, select Install from repository.

Go to Flow FavManager Repository → Program add-ons → Flow FavManager and install.

VERY IMPORTANT: You must type the address exactly. DO NOT copy the URL from your browser bar (github.com...), as it will not work in Kodi and will result in an empty folder. If a security warning appears, go to Settings and enable "Unknown sources".

Manual Installation via ZIP

If you prefer to download the file manually from GitHub.

Download the repository.flowfavmanager-1.0.0.zip file from the Releases section on GitHub.

In Kodi, go to Add-ons → Box icon → Install from zip file.

Locate the downloaded file on your device and select it to complete the installation.

Language support: If the addon is not in your preferred language after installation, check the Releases section for more language options.
